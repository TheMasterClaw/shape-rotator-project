## Production Notes

### Screen Recording Tips
- Use clean browser profile (no bookmarks visible)
- Hide notifications
- Zoom in for important interactions
- Use cursor highlight for clicks

### Voiceover Tips
- Record in quiet environment
- Use pop filter
- Speak slightly slower than normal
- Pause between major sections

### Post-Production
- Add background music (low volume, ambient)
- Use smooth transitions between scenes
- Add captions for accessibility
- Export at 1080p, 60fps

### B-Roll Ideas
- Abstract network animations
- Code scrolling (aesthetic)
- Contract verification on BaseScan
- Discord reactions/emojis for problem statement
